## 项目说明

学习代码在`sean-dev`分支

将原项目中每一讲的`tutorial0x`和`tutorial0x_answer`合并为`leptjson0x`, 即每一讲的课程代码和作业代码共用一套代码

目前学习内容以注释的形式记录在代码中,  后续考虑增加学习笔记
